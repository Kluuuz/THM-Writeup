# TryHackMe Write-ups

This repository contains my personal write-ups for TryHackMe rooms that I've completed. The purpose of these write-ups is to document my learning, practice my methodology, and keep track of the techniques and tools used throughout each room.

## Repository Structure

Each room has its own folder containing:

* `ctf.md` – The walkthrough
* `pictures/` – Screenshots used in the write-up

## Disclaimer

These write-ups are for educational purposes only and are based on intentionally vulnerable environments provided by TryHackMe. Please use these techniques responsibly and only on systems you are authorized to test.
